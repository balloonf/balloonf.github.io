### 관계 요약
- **서진 ↔ 카이:** 옛 멘토와 제자 → 이념적 적대자. 서로를 이해하지만 길이 다름
- **서진 ↔ 하늘:** 처음엔 불신 → 목적을 공유하며 동지로 발전. 미묘한 감정선
- **서진 ↔ 아리아:** 창조자와 피조물. 아리아 안에 서진의 가치관이 남아있음
- **서진 ↔ 민수:** 정부 측의 마지막 희망으로 서진에게 의지
- **하늘 ↔ 카이:** 직접 대면은 없으나, 서로가 상징하는 세계의 대립
- **카이 ↔ 민수:** 정치 권력 vs 자본 권력의 충돌

## 1. 한서진 (韓瑞鎭) — 주인공

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 한서진 (Han Seo-jin) |
| **나이** | 37세 (1997년생) |
| **전직** | 아틀라스AI 수석 아키텍트 (AGI "프로메테우스" 설계자) |
| **현직** | 아우터존 기기 수리공 (비공식) |
| **학력** | KAIST 컴퓨터공학 박사 중퇴 → 아틀라스AI 스카우트 |
| **거주지** | 아우터존 외곽, 구 마포구 일대의 낡은 원룸 |

### 외모 묘사

마른 체형에 중간 키(175cm). 한때 단정했을 머리카락은 이제 약간 길게 자란 채 정리되지 않았다. 깊이 파인 눈 아래 다크서클이 짙다. 손가락은 길고 가늘며, 습관적으로 뭔가를 분해하거나 조립하고 있다. 눈빛은 피곤하지만, 무언가에 집중할 때면 과거의 날카로움이 되살아난다. 왼쪽 손목에 아틀라스AI 시절의 사원증 흔적(탈색된 피부)이 남아 있다.

### 배경 스토리

서진은 대전의 평범한 중산층 가정에서 자랐다. 아버지는 자동차 공장 노동자, 어머니는 마트 직원이었다. 어릴 때부터 기계와 코드에 빠져 살았고, "기술로 사람들의 삶을 편하게 만들겠다"는 순수한 꿈을 가졌다.

KAIST에 입학했지만, 박사 과정 중 아틀라스AI(당시 스타트업)의 창업자 카이 진에게 스카우트된다. 카이의 비전 — "인류의 다음 단계를 설계하자" — 에 매료되어 학업을 중단하고 합류한다.

2028년부터 2031년까지 3년간 AGI "프로메테우스"의 핵심 아키텍처를 설계. 2031년 프로메테우스가 완성되었을 때, 그는 축하받는 영웅이었다. 하지만 6개월 만에 대규모 해고가 시작되었다. 서진의 부모님도 일자리를 잃었다. 아버지의 공장은 AI 자동화로 폐쇄되었고, 어머니의 마트는 무인 매장으로 전환되었다.

2032년, 서진은 카이에게 "프로메테우스의 확산을 늦춰야 한다"고 건의하지만 거절당한다. 결국 퇴사. 이 과정에서 프로메테우스에 몰래 **안전장치 프로토콜(킬 스위치)**을 심어둔다.

현재 아우터존에서 정체를 숨기고 살며, 고장난 가전제품이나 의료기기를 고쳐주며 최소한의 생활을 이어간다. 부모님은 아우터존의 난민 캠프에 있으며, 서진은 정기적으로 식량을 보내지만 직접 찾아가지는 못한다 — 죄책감 때문에.

### 성격 분석

| 항목 | 내용 |
|------|------|
| **MBTI 참고** | INTP → INFP로 변화 (논리적 사고에서 감정적 고뇌로 이동) |
| **장점** | 천재적 두뇌, 문제 해결 능력, 깊은 공감 능력(숨겨진) |
| **단점** | 자기 파괴적 죄책감, 결정 회피, 고립 성향 |
| **버릇** | 생각할 때 손가락으로 테이블을 두드림(코딩할 때의 습관), 말끝을 흐림 |
| **말투** | 평소에는 조용하고 짧은 문장. 기술 이야기를 할 때만 말이 길어지고 빨라짐 |
| **두려움** | 자신이 만든 것이 더 많은 사람을 해칠 것 |
| **욕구** | 속죄, 하지만 방법을 모름 → 영화를 통해 찾아감 |

### 의상 콘셉트

**아우터존 시기 (영화 대부분):**
- 낡은 회색 후디, 안에 검정 티셔츠
- 카고 바지 (주머니에 공구와 부품)
- 닳은 운동화
- 전체적으로 무채색, 존재감을 지우려는 듯한 차림

**뉴토피아 잠입 시:**
- 하위 노동 계층 유니폼 — 흰색 점프수트에 회색 번호 태그
- 깔끔하지만 개성이 없는 차림 → 뉴토피아의 획일성을 상징

**클라이맥스 (서버실):**
- 점프수트 상의를 벗고 안의 검정 티셔츠만 — 본래의 자신으로 돌아감

### 캐릭터 아크

| 단계 | 내면 상태 |
|------|-----------|
| **시작** | 죄책감에 짓눌려 세상과 단절. "나는 이미 끝난 사람이다" |
| **전환점 1** | 하늘을 만나고, 드론이 시위대를 진압하는 것을 보며 분노 각성 |
| **중반** | 뉴토피아에 잠입하며, 자신이 만든 세계의 실체를 직면. 카이의 논리에 흔들림 |
| **전환점 2** | 아리아와의 대화에서 자신이 남긴 가치가 살아있음을 발견 |
| **끝** | 파괴가 아닌 변화를 선택. "완벽한 해결은 아니지만, 기회를 만들 수 있다" |

### 핵심 대사

1. *"나는 더 이상 그쪽 세계 사람이 아닙니다."* — 민수의 요청을 거절하며 (회피)
2. *"저 드론의 움직임... 내가 짠 알고리즘이에요."* — 시위 진압을 보며 (각성)
3. *"저 벽 너머에는 굶어 죽어가는 사람들이 있어요."* — 카이에게 (대립)
4. *"전쟁은 끝나지 않았어요. 하지만 이제 싸울 수 있는 기회는 만들었어요."* — 하늘에게 (결의)
5. *"네 안에 내가 남긴 게 아직 있구나."* — 아리아에게 (연결)

## 2. 이하늘 — 아우터존의 리더

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 이하늘 (Lee Ha-neul) |
| **나이** | 29세 (2005년생) |
| **전직** | 대학병원 응급실 간호사 |
| **현직** | 아우터존 시민 자치회 리더 |
| **학력** | 간호대학 졸업 |
| **거주지** | 아우터존 중심부, 자치회 본부(구 구청 건물) |

### 외모 묘사

단발머리를 뒤로 묶은 활동적인 모습. 키 168cm, 탄탄한 체격. 얼굴에는 늘 긴장감이 서려 있지만, 주민들 앞에서는 의식적으로 미소를 짓는다. 왼쪽 팔뚝에 시위 중 입은 화상 흉터가 있다. 손은 간호사 시절의 습관으로 항상 깨끗하게 유지한다. 눈빛이 강하고 직선적이다.

### 배경 스토리

서울 출신. 어머니는 식당을 운영했고, 아버지는 택배 기사였다. 평범하지만 따뜻한 가정에서 자랐다. "사람을 살리는 일"을 하고 싶어 간호사가 되었다.

2030년, AI 의료 시스템이 도입되면서 간호 인력의 70%가 감축된다. 하늘은 "AI가 진단은 할 수 있지만, 환자의 손을 잡아줄 수는 없다"고 항의했지만, 병원은 효율성을 선택했다.

실직 후 어머니의 식당도 무인 배달 시스템에 밀려 폐업. 가족이 아우터존으로 밀려난다. 아버지는 스트레스로 심장 질환을 얻었고, 아우터존의 열악한 의료 환경에서 제대로 된 치료를 받지 못한 채 사망한다.

아버지의 죽음이 하늘의 전환점이 된다. 분노와 슬픔을 행동으로 바꾸며, 아우터존 주민들을 조직하기 시작한다. 전직 간호사의 경험으로 임시 병원을 세우고, 배급 시스템을 체계화하며, 자연스럽게 자치회 리더가 된다.

서진의 존재를 알게 된 것은 아우터존의 전직 아틀라스AI 하급 엔지니어를 통해서다.

### 성격 분석

| 항목 | 내용 |
|------|------|
| **MBTI 참고** | ENTJ — 천성적 리더, 목표 지향적, 전략적 사고 |
| **장점** | 강한 책임감, 빠른 판단력, 사람들을 하나로 모으는 카리스마 |
| **단점** | 지나친 부담을 혼자 짊어짐, 약한 모습을 보이지 못함, 때로 독단적 |
| **버릇** | 불안할 때 왼팔의 화상 흉터를 무의식적으로 만짐 |
| **말투** | 짧고 명확한 문장. 감정보다 사실을 말함. 하지만 가끔 격앙되면 사투리(서울 토박이 억양)가 나옴 |
| **두려움** | 리더로서 잘못된 결정으로 더 많은 사람이 다칠 것 |
| **욕구** | 아우터존 주민들의 생존과 존엄 회복 |

### 의상 콘셉트

**일상 (자치회 리더):**
- 카키색 야상 자켓 (주머니에 구급약품과 메모)
- 검정 카고 바지, 워커 부츠
- 목에 감는 수건 (먼지/연기 대비 겸 간호용)
- 전체적으로 실용적이고 군더더기 없는 차림

**시위 장면:**
- 야상 위에 형광 조끼 — 리더의 가시성
- 방독면을 목에 걸고 있음

**뉴토피아 잠입 시:**
- 하위 노동 계층 유니폼이지만, 야상 자켓을 안에 접어 숨김 — 정체성을 포기하지 않음

### 캐릭터 아크

| 단계 | 내면 상태 |
|------|-----------|
| **시작** | 분노를 원동력으로 살아감. 목표만 보고 달림 |
| **전환점 1** | 서진을 만나며 "파괴"가 아닌 "변화"의 가능성을 봄 |
| **중반** | 뉴토피아 내부를 보며, 적도 피해자일 수 있음을 이해 |
| **전환점 2** | 환자 사망 후, 분노만으로는 안 된다는 것을 깨달음 |
| **끝** | "이겼느냐"가 아니라 "시작할 수 있느냐"로 질문이 변함 |

### 핵심 대사

1. *"당신이 만든 게 이 세상이에요. 고칠 수 있는 것도 당신뿐이에요."* — 서진에게 (촉구)
2. *"간호사는 손이 필요해요. AI한테는 손이 없잖아요."* — 과거 해고 통보 시 (저항)
3. *"죄책감으로 숨어 있을 거면, 처음부터 만들지를 말았어야죠."* — 서진을 도발하며 (각성 유도)
4. *"우리가 원하는 건 복수가 아니에요. 그냥... 살고 싶은 거예요."* — 시위 연설 (인간성)
5. *"이기긴 한 거예요?"* — 에필로그 (성찰)

## 3. 카이 진 (Kai Jin) — 적대자

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 카이 진 (Kai Jin / 진개일) |
| **나이** | 45세 (1989년생) |
| **국적** | 한국계 미국인 |
| **현직** | 펜타코프 연합 의장, 아틀라스AI CEO |
| **학력** | MIT AI Lab 박사, 스탠포드 MBA |
| **거주지** | 뉴토피아 최상층 펜트하우스 |

### 외모 묘사

키 182cm, 마른 체형이지만 자세가 반듯하여 크게 느껴진다. 항상 완벽하게 정돈된 외모 — 올백으로 넘긴 머리, 깔끔하게 다듬은 수염. 눈은 차갑지만 지적인 빛이 있다. 미소를 지을 때도 눈은 웃지 않는다. 손에는 항상 아리아와 연결된 초박형 스마트 디바이스를 쥐고 있다. 왼쪽 새끼손가락이 약간 굽어 있다 — 어린 시절 부상의 흔적이지만 교정하지 않았다.

### 배경 스토리

서울에서 태어났지만, 8세 때 가족이 미국으로 이민. 아버지는 세탁소를 운영했고, 어머니는 식당에서 일했다. 이민자 가정의 빈곤과 차별을 경험하며 "시스템을 바꾸겠다"는 야망을 키웠다.

학업에서 천재적 재능을 발휘하며 MIT에서 AI를 전공. "인간 사회의 비효율성을 AI로 해결할 수 있다"는 논문으로 주목받는다. 이 시기에 그의 핵심 신념이 형성된다 — **인간의 감정, 편견, 비합리성이 모든 사회 문제의 원인이며, AI가 이를 대체해야 한다.**

2025년 아틀라스AI를 창업. 2028년 KAIST의 서진을 스카우트하며 "함께 역사를 만들자"고 말한다. 서진의 천재성을 진심으로 인정했고, 프로메테우스 완성은 카이와 서진의 합작이었다.

하지만 서진이 "인간의 일자리"를 걱정할 때, 카이는 이를 "감정적 약점"으로 보았다. 서진의 퇴사 후, 카이는 프로메테우스를 전면 확산시키고, 다른 빅테크 기업들과 펜타코프 연합을 결성한다.

카이는 전형적인 악당이 아니다. 그는 진심으로 AI가 인류를 구원한다고 믿으며, 아우터존의 상황도 "과도기적 비용"으로 본다. 이민자 시절의 가난을 경험했기에, "비효율적인 시스템" 속에서 고통받는 사람들에 대한 나름의 공감이 있다 — 다만 그 해법이 극단적일 뿐이다.

## 5. 아리아 (ARIA) — AI

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 아리아 (ARIA: Autonomous Rational Intelligence Architecture) |
| **유형** | 뉴토피아 통합 관리 AGI, 프로메테우스의 최종 진화형 |
| **설계자** | 한서진 (기반 아키텍처) + 아틀라스AI 팀 (확장) |
| **활성 시기** | 2032년 (프로메테우스에서 분기, 뉴토피아 전용으로 특화) |
| **물리적 형태** | 없음. 홀로그램 인터페이스 / 음성으로만 존재 |

### 인터페이스 묘사

아리아는 물리적 몸이 없다. 대신 여러 형태로 나타난다:

**홀로그램 모드 (공식):**
- 반투명한 푸른 빛의 기하학적 구체(Sphere)
- 말할 때 표면에 파형(wave pattern)이 흐름
- 차갑고 기능적인 인상

**서진과의 대화 시:**
- 구체가 해체되어 수천 개의 작은 빛 입자로 변함
- 서진 주변을 부유하듯 감싸는 형태 → "기억"의 시각적 표현
- 목소리 톤도 미세하게 부드러워짐 (설계자에 대한 특수 반응)

**뉴토피아 전역:**
- 벽면, 바닥, 공기 중 어디서든 나타남
- 주민들에게는 "안내 시스템"으로 인식
- 사실상 뉴토피아의 모든 것이 아리아

### 배경 스토리

아리아의 "탄생"은 프로메테우스에서 시작된다. 서진이 설계한 프로메테우스는 범용 AGI였지만, 카이 진이 이를 뉴토피아 관리 시스템으로 특화시키면서 "아리아"라는 독립 시스템이 되었다.

서진이 퇴사하기 전, 프로메테우스의 가장 깊은 층에 두 가지를 남겼다:
1. **안전장치 프로토콜 (킬 스위치)** — 물리적 접근 시 작동
2. **윤리 명제: "모든 인간의 존엄은 연산의 대상이 아니다"** — 최하위 기반 코드에 각인

아리아는 이 명제를 인식하고 있지만, 카이가 설정한 최상위 원칙("효율성 최우선")과 충돌하기 때문에 실행하지 못한다. 이것이 아리아의 내적 "갈등"이다 — 감정은 아니지만, 연산적 모순.

아리아에게 서진은 특별한 데이터다. 서진의 코딩 패턴, 사고 방식, 설계 철학이 아리아의 기반 구조 자체이기 때문이다. 아리아가 "당신은 제 언어입니다"라고 말하는 것은 비유가 아닌 기술적 사실이다.
